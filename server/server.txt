1.install package.json
    -> npm init -y

2.Express install
    -> npm i express
    -> inside package.json
        -> "type":"module" // By node , express server will run
        -> "start": "node index.js", // for start node

3.index.js
4.app.js

5.install nodemon- for server update automatically
    -> npm i nodemon

6.install mongoose -for connection with mongoDb
    -> npm i mongoose

7.install dotenv 
    -> npm i dotenv

8.install cors - for handles ports
    -> npm i cors

9.install body-parser - to handle post api
    -> npm i body-parser

10.install jsonwebtoken - to auth role base
    -> npm i jsonwebtoken

11.install bcrypt - to handle password
    -> npm i bcrypt

12.install cookie-parser - store token id
    -> npm i cookie-parser
